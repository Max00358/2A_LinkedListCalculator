The goal of this project is to implement a simple calculator that can add and subtract variables stored in a linked list. 

  Sample Input 1:
```
CRT 2
DEF x 3
DEF y 2
DEF z 3
DEF a 4
REM z
ADD x y z
ADD x y x
PRT x
END
```
  Sample Output 1:
  ```
success
success
success
failure
failure
failure
failure
success
5
  ```
  Sample Input 2:
  ```
CRT 10
DEF x 4
DEF y 5
DEF z 6
ADD x y z
PRT z
END
  ```
  Sample Output 2:
  ```
success
success
success
success
success
9
  ```
